## 📦 安装

### 一键安装

点击上方徽章，或在浏览器中直接打开此链接：

[https://raw.githubusercontent.com/Okamitimo233/linux-do-catppuccin/main/catppuccin.user.less](https://raw.githubusercontent.com/Okamitimo233/linux-do-catppuccin/main/catppuccin.user.less)

Stylus 会自动检测到用户样式并弹出安装对话框。点击 **"Install style"** 即可完成！

> [!TIP]
> 请先安装 [Stylus](https://github.com/openstyles/stylus) 浏览器扩展：  
> [Chrome Web Store](https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne) · [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/styl-us/)

### 手动安装

1. 安装 [Stylus](https://github.com/openstyles/stylus) 扩展
2. 打开 Stylus 弹出窗口 → 点击 **管理**
3. 点击 **编写新样式** → 勾选 **作为 UserCSS**
4. 粘贴 [`catppuccin.user.less`](./catppuccin.user.less) 的全部内容
5. 点击 **保存**

## 🔧 配置

安装后，打开 Stylus 弹出窗口，点击 **Linux DO Catppuccin** 旁边的齿轮图标，即可调整：

- **浅色风味** — 系统/浏览器处于浅色模式时使用的调色板
- **深色风味** — 深色模式下使用的调色板
- **强调色** — 链接、按钮和交互元素的突出颜色

修改即时生效，无需刷新页面。

## 🧩 特性

- 🌗 通过 `prefers-color-scheme` 自动检测浅色/深色模式
- 🎯 Discourse 主题检测（支持 `data-theme`、`.theme-dark`、`.dark` 等属性）
- 🎨 完整的 Catppuccin 调色板：4 种风味 × 15 种强调色
- 🖌️ 覆盖范围：顶栏、侧边栏、话题列表、帖子、按钮、输入框、弹窗、代码块、滚动条
- 🔄 通过 Stylus 支持自动更新
