# Linux DO Catppuccin

Soothing pastel theme for [Linux DO](https://linux.do).

## Installation

Install the userstyle with the Stylus browser extension:

[![Install directly with Stylus](https://img.shields.io/badge/Install%20directly%20with-Stylus-238b8b?style=for-the-badge)](https://raw.githubusercontent.com/YOUR_USERNAME/linux-do-catppuccin/main/catppuccin.user.less)

Or manually:

1. Install the [Stylus](https://github.com/openstyles/stylus) extension for your browser.
2. Open `catppuccin.user.less` and click "Install Style".
3. Configure the theme (light/dark flavor and accent color) in Stylus settings.

## Usage

After installation, the theme will automatically be applied when visiting [linux.do](https://linux.do). Use the Stylus popup to adjust:

- **Light Flavor** – Latte, Frappé, Macchiato, Mocha
- **Dark Flavor** – Latte, Frappé, Macchiato, Mocha
- **Accent Color** – Rosewater, Flamingo, Pink, Mauve, Red, Maroon, Peach, Yellow, Green, Teal, Blue, Sapphire, Sky, Lavender, Gray
